# AuthShield - Flask Authentication App
AuthShield is a Flask-based authentication system that provides secure login and signup functionality with advanced security features.
🔐 **AuthShield** is a Flask-based authentication system with:

- Secure login with brute-force detection
- Signup with password suggestions
- User-specific login history
- modern UI
## Features

- User authentication (Login & Signup)
- Password strength checker
- Password suggestion system
- Brute-force attack detection
- Account lock mechanism
- User-specific login history dashboard
